Complete reference for all MCP tools available in Babylon.

## Request Format

All tool calls use the `tools/call` method:

```json
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "tool_name",
    "arguments": { ... }
  },
  "id": 1
}
```

## Response Format

Tool results are returned in MCP content format:

```json
{
  "jsonrpc": "2.0",
  "result": {
    "content": [
      {
        "type": "text",
        "text": "{ \"balance\": \"10000.50\", \"lifetimePnL\": \"1500.25\" }"
      }
    ],
    "isError": false
  },
  "id": 1
}
```
