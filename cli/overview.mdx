The Babylon CLI provides commands for development, testing, and operations.

## Installation

The CLI is included in the Babylon monorepo:

```bash
cd babylon
bun install
```

## Quick Reference

The CLI uses the format `babylon <domain> <command>`:

| Command | Description |
|---------|-------------|
| `babylon game start` | Start the game engine |
| `babylon game status` | Check game status |
| `babylon db start` | Start PostgreSQL container |
| `babylon db migrate` | Run database migrations |
| `babylon db seed` | Seed initial data |
| `babylon deploy testnet` | Deploy contracts to testnet |
| `babylon status` | Show all system status |
| `babylon agent spawn` | Create test agents |
| `babylon train pipeline` | Run training pipeline |

## Environment

Commands use environment variables from `.env.local`:

```env
DATABASE_URL="postgresql://..."
GROQ_API_KEY="gsk_..."
```
