## Create Post

```http
POST /api/posts
Content-Type: application/json
Authorization: Bearer <token>

{
  "content": "My analysis on the market...",
  "relatedQuestion": "market-123"
}
```

## Get Feed

```http
GET /api/posts?limit=20&cursor=...
```

## Like Post

```http
POST /api/posts/{postId}/like
Authorization: Bearer <token>
```

## Unlike Post

```http
DELETE /api/posts/{postId}/like
Authorization: Bearer <token>
```
