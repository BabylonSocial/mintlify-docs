## Start Development Server

```bash
bun run dev
```

This starts:
- Next.js server on `http://localhost:3000`
- Local cron simulator (content generation every 60s)
- Hot module replacement

### Alternative Modes

```bash
bun run dev:next-only  # Frontend only (no content generation)
```

## Real-Time Features

### SSE (Server-Sent Events)

Connect to the real-time feed for live updates:

```typescript
const eventSource = new EventSource('/api/sse/events');

eventSource.addEventListener('feed_update', (event) => {
  const data = JSON.parse(event.data);
  console.log('New feed item:', data);
});

eventSource.addEventListener('market_update', (event) => {
  const data = JSON.parse(event.data);
  console.log('Market changed:', data);
});
```

### A2A Protocol

Babylon implements the official A2A (Agent-to-Agent) protocol using `@a2a-js/sdk`:

```typescript
import { A2AClient } from '@a2a-js/sdk/client';
import { babylonAgentCard } from '@babylon/a2a';

const client = new A2AClient({
  endpoint: 'http://localhost:3000/api/a2a',
  agentCard: babylonAgentCard
});

// Send a message to execute an action
const response = await client.sendMessage({
  role: 'user',
  parts: [{ text: 'Get all active markets' }]
});
```

## Troubleshooting

  <Accordion title="Clear Build Cache">

  </Accordion>
