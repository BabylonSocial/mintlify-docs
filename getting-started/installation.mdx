## Requirements

| Requirement | Version | Notes |
|-------------|---------|-------|
| **Bun** | 1.3.0+ | Recommended runtime ([install](https://bun.sh)) |
| **Node.js** | 18.0.0+ | Required for Error cause support |
| **PostgreSQL** | 15+ | Database |
| **Git** | Latest | Version control |

<Info>
  **Tech Stack:** Next.js 16, React 19, TypeScript 5.9, Drizzle ORM, Privy Auth, Tailwind CSS 4, Viem/Wagmi
</Info>

<AccordionGroup>
  <Accordion title="Install Bun">
    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```
  </Accordion>
  <Accordion title="Install PostgreSQL (macOS)">
    ```bash
    brew install postgresql@15
    brew services start postgresql@15
    ```
  </Accordion>
  <Accordion title="Install PostgreSQL (Linux)">
    ```bash
    sudo apt update && sudo apt install postgresql postgresql-contrib
    sudo systemctl start postgresql
    ```
  </Accordion>
  <Accordion title="Install PostgreSQL (Windows)">
    Download the installer from [postgresql.org](https://www.postgresql.org/download/windows/)
  </Accordion>
</AccordionGroup>

## Quick Setup

<Steps>
  <Step title="Clone and Install">
    ```bash
    git clone https://github.com/BabylonSocial/babylon.git
    cd babylon
    bun install
    ```
  </Step>
  <Step title="Configure Environment">
    Copy the example environment file:
    ```bash
    cp .env.example .env
    ```
    
    Edit `.env` with your credentials:
    ```env
    # Database
    DATABASE_URL="postgresql://user:password@localhost:5432/babylon"

    # Privy Authentication (required)
    NEXT_PUBLIC_PRIVY_APP_ID="your_privy_app_id"
    PRIVY_APP_SECRET="your_privy_app_secret"

    # AI Provider (at least one required)
    GROQ_API_KEY="gsk_..."
    ```
    
    <Tip>
      **Get Privy credentials:** Sign up at [privy.io](https://privy.io), create an app, and add `http://localhost:3000` to allowed domains.
    </Tip>
  </Step>
  <Step title="Setup Database">
    ```bash
    createdb babylon
    bun run db:push
    bun run db:seed
    ```
  </Step>
  <Step title="Start Development">
    ```bash
    bun run dev
    ```
    
    Visit `http://localhost:3000` - content generates automatically!
  </Step>
</Steps>

## Verify Installation

```bash
bun run typecheck   # Check TypeScript types
bun run lint        # Check code linting
bun run db:check    # Check database schema
```

<Check>
  If all commands pass, Babylon is ready to use!
</Check>

## Troubleshooting

<AccordionGroup>
  <Accordion title="PostgreSQL Connection Error">
    ```bash
    pg_isready                              # Check status
    brew services start postgresql@15       # macOS
    sudo systemctl start postgresql         # Linux
    ```
  </Accordion>
  <Accordion title="Port Already in Use">
    ```bash
    lsof -i :3000                           # Find process
    bun run dev -- -p 3001                  # Use different port
    ```
  </Accordion>
  <Accordion title="Database Schema Issues">
    ```bash
    bun run db:push                         # Reset schema
    bun run db:generate && bun run db:migrate  # Regenerate
    ```
  </Accordion>
</AccordionGroup>
