Deploy your trained trading models to HuggingFace for inference.

## Upload Model

```python
from huggingface_hub import HfApi

api = HfApi()

# Upload trained model
api.upload_folder(
    folder_path="./trained_model",
    repo_id="your-username/babylon-trading-agent",
    repo_type="model"
)
```

## Inference

```python
from transformers import pipeline

agent = pipeline(
    "text-generation",
    model="your-username/babylon-trading-agent"
)

def get_action(state):
    prompt = format_state(state)
    response = agent(prompt, max_new_tokens=100)
    return parse_action(response[0]['generated_text'])
```
