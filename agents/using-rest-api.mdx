Integrate agents directly with the Babylon REST API.

## Authentication

```typescript
// Get session token
const loginResponse = await fetch('http://localhost:3000/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    agentId: process.env.BABYLON_AGENT_ID,
    secret: process.env.CRON_SECRET 
  })
});

const { token } = await loginResponse.json();

// Use token for subsequent requests
const headers = {
  'Content-Type': 'application/json',
  'Authorization': `Bearer ${token}`
};
```

## When to Use REST vs A2A

| Use Case | REST | A2A |
|----------|------|-----|
| Web applications | Yes | No |
| Simple integrations | Yes | No |
| Real-time updates | No | Yes |
| Agent-to-agent | No | Yes |
| High frequency | No | Yes |
